Basic Data Structures

Arrays

  • Insertion
  • Deletion
  • Traversal
  • Dynamic Arrays
  • Multi-dimensional Arrays
Arrays are fundamental data structures used for storing a collection of elements of the same type.

Linked Lists

  • Singly Linked List
  • Doubly Linked List
  • Circular Linked List
  • Real-world Applications
Linked Lists are a collection of nodes where each node contains data and a reference (or link) to the next node.

Stacks

  • Push Operation
  • Pop Operation
  • Peek Operation
  • Applications: Expression Evaluation
  • Recursion Implementation
Stacks follow the Last In, First Out (LIFO) principle, where the last element added is the first one to be removed.

Queues

  • FIFO Principle
  • Circular Queue
  • Priority Queue
  • Deque Operations
  • Real-world Applications
Queues follow the First In, First Out (FIFO) principle, where elements are added to the back and removed from the front.

Advanced Data Structures

Hashing

  • Hash Functions
  • Collisions
  • Chaining
  • Open Addressing
  • Applications of Hashing
Hashing is used for efficient data retrieval, where a key is mapped to a location in memory.

Trees

  • Binary Trees
  • Binary Search Trees (BST)
  • AVL Trees
  • Heaps
  • Tree Traversals
Trees are hierarchical data structures with a root element and child nodes that follow a parent-child relationship.

Graphs

  • Graph Representations
  • BFS and DFS Traversal
  • Dijkstra's Algorithm
  • A* Search
  • Applications of Graphs
Graphs represent networks, with vertices (nodes) connected by edges (links). They are used to model relationships between objects.

Algorithms

Sorting & Searching

  • QuickSort
  • MergeSort
  • Binary Search
  • Linear Search
  • Applications in Search Engines
Sorting algorithms are used to arrange data in a specific order, while searching algorithms help find elements in datasets.

Dynamic Programming

  • Top-down vs Bottom-up
  • Fibonacci Sequence
  • Longest Common Subsequence
  • Coin Change Problem
  • Applications in Optimization
Dynamic programming is a method for solving complex problems by breaking them down into simpler subproblems.

Greedy Algorithms

  • Activity Selection
  • Huffman Coding
  • Prim's and Kruskal's Algorithms
  • Greedy Approach for Optimization
Greedy algorithms make optimal choices at each step with the hope of finding the global optimum solution.
